# Cafeteria Management System (C)

A simple console-based Cafeteria Management System developed in C language.  
This project helps manage cafeteria orders, menu items, and billing operations efficiently.

## Features
- Display food menu
- Take customer orders
- Generate bills
- Calculate total price
- Manage cafeteria operations

## Technologies Used
- C Programming Language
- Dev-C++
- File Handling

## Purpose
This project was developed as a university semester project to improve programming concepts, logical thinking, and file handling in C language.

## How to Run
1. Open the project in Dev-C++ or any C compiler.
2. Compile the source code.
3. Run the executable file.
